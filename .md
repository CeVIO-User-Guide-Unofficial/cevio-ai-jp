<h1>メニュー</h1>
<br/>
<div class="main_text">
<div><img src='../../image/menu_01.png' border='0'/></div>

<h2>ファイル</h2>
<b><font color='#000000'>新規プロジェクト</font></b><br/>
新たなプロジェクト（トークやソングなどの全容）を作成します。<br/>
編集中の内容がある場合は、保存の確認画面が表示されます。<br/>
<br/>
<b><font color='#000000'>開く</font></b><br/>
保存したプロジェクトファイルを開いて作業を再開します。<br/>
編集中の内容がある場合は、保存の確認画面が表示されます。<br/>
<br/>
<b><font color='#000000'>バックアップを開く</font></b><br/>
プロジェクトのバックアップファイルを開きます。<br/>
バックアップファイルは5分間隔で自動保存され、5日間保存後に削除されます。<br/>
<br/>
<b><font color='#000000'>保存</font></b><br/>
編集中のプロジェクトファイルを上書き保存します。<br/>
<br/>
<b><font color='#000000'>名前を付けて保存</font></b><br/>
編集中のプロジェクトに名前を付けて保存します。<br/>
ファイル名を入力してから［保存］をクリックしてください。保存したファイルの拡張子は「.ccs」になります。<br/>
<br/>
<b><font color='#000000'>インポート</font></b><br/>
<blockquote style='margin:0 0 0 40px;padding:0px;border:currentcolor'>
<b><font color='#000000'>セリフのテキストの読み込み</font></b><br/>
トークトラックにテキストファイルのセリフを読み込みます。<br/>
<a href='../../talktrack/talk_04'>セリフのインポート</a><br/>
<br/>
<b><font color='#000000'>青空文庫の読み込み</font></b><br/>
トークトラックに青空文庫形式のテキストファイルを（注釈文［＃…］を除外して）読み込みます。<br/>
<br/>
<b><font color='#000000'>字幕の読み込み</font></b><br/>
トークトラックに、SubRip形式（.srt）または、SubViewer形式（.sbv）の字幕ファイルを読み込みます。<br/>
<br/>
<b><font color='#000000'>MIDIの読み込み</font></b><br/>
ソングトラックにスタンダードMIDIファイルを読み込みます。<br/>
<a href='https://cevio.jp/guide/cevio_ai/songtrack/fileimport/'>MIDI／MusicXML</a><br/>
<br/>
<b><font color='#000000'>MusicXMLの読み込み</font></b><br/>
ソングトラックにMusicXMLファイルを読み込みます。<br/>
<br/>
<b><font color='#000000'>オーディオファイルの読み込み</font></b><br/>
外部オーディオトラックのポジションカーソルの位置にオーディオファイルを読み込みます。<br/>
<a href='https://cevio.jp/guide/cevio_ai/audio/'>オーディオファイルの形式について</a><br/>
<br/>
<b><font color='#000000'>トラック単位の読み込み</font></b><br/>
「トラック単位の保存」で保存したファイルを読み込みます。<br/>
<br/>
<b><font color='#000000'>設定の読み込み</font></b><br/>
保存済みの環境設定ファイルを読み込みます。<br/>
設定ファイルに含まれる「全般設定」「ユーザー辞書」「プリセット」「セリフクリップ」から選択して読み込み可能です。「ユーザー辞書」「プリセット」「セリフクリップ」は、「既存内容を置き換える」か「既存内容に追加する」かを選べます。<br/>
</blockquote>
<br/>
<b><font color='#000000'>エクスポート</font></b><br/>
<blockquote style='margin:0 0 0 40px;padding:0px;border:currentcolor'>
<b><font color='#000000'>ミックスダウンWAV書き出し</font></b><br/>
全トラックをまとめてステレオWAV形式（48kHz 16bit）で書き出します。<br/>
<br/>
<div><b><font color='#000000'>セリフの連続WAV書き出し</font></b></div>
選択したトークトラックの内容を、セリフごとにモノラルWAV形式で書き出します。<br/>
<div><img src='../../image/Export_AudioFile_PerEachLine.png' border='0'/></div>
※WAVファイルのサンプリングレートとビット深度は、<a href='https://cevio.jp/guide/cevio_ai/option/'>オプション</a>で指定可能。<br/>
※「詳細」の「トラック毎にフォルダと通し番号を分ける」をオンにすると、トラック毎にフォルダと通し番号を分けます。初期設定はオフで、フォルダを分けず全トラック全セリフの通し番号になります。<br/>
※ファイル名が意図せず固定されてしまったときは、「ファイル名パターン」を「$通番$_$キャスト名$_$セリフ$」にすると初期設定に戻ります。<br/>
※「セリフをテキスト出力」がオンのとき、セリフ内容をテキストファイルに同時に出力します。<br/>
※「タイミング情報をテキスト出力」がオンのとき、音素毎の発声タイミングをLab形式（.lab）で同時に出力します。対応するソフトで口の動きの同期などに利用できます。<br/>
<br/>
<div><b><font color='#000000'>セリフのテキスト書き出し</font></b></div>
選択したトークトラックのセリフを、カンマ区切りテキスト形式で書き出します。<br/>
<br/>
<div><b><font color='#000000'>字幕テキストの書き出し</font></b></div>
選択したトークトラックのセリフを、SubRip形式（.srt）または、SubViewer形式（.sbv）で書き出します。YouTubeの字幕などに利用できます。<br/>
<br/>
<div><b><font color='#000000'>ソングのWAV書き出し</font></b></div>
選択したソングトラックをモノラルWAV形式で書き出します。<br/>
※WAVファイルのサンプリングレートとビット深度は、<a href='https://cevio.jp/guide/cevio_ai/option/'>オプション</a>で指定可能。<br/>
<br/>
<b><font color='#000000'>MIDIの書き出し</font></b><br/>
選択したソングトラックをMIDI形式で書き出します。<br/>
※複数トラック選択時も1ファイルにまとめられ、調号は最初の選択トラックのみ「Conductor Track」に反映されます。<br/>
<br/>
<b><font color='#000000'>MusicXMLの書き出し</font></b><br/>
選択したソングトラックをMusicXML形式で書き出します。<br/>
<br/>
<b><font color='#000000'>トラック単位の保存</font></b><br/>
選択したトラックの編集内容を保存します。<br/>
保存したファイルは、別のプロジェクトにインポートして再利用できます。<br/>
保存したファイルの拡張子は「.ccst」になります。<br/>
<br/>
<b><font color='#000000'>設定の書き出し</font></b><br/>
現在の環境設定ファイルを書き出します。<br/>
「全般設定」「ユーザー辞書」「プリセット」「セリフクリップ」の全部、または一部選択して書き出し可能です。<br/>
保存したファイルの拡張子は「.ccsa」になります。<br/>
</blockquote>
<br/>
<b><font color='#000000'>最近使ったプロジェクト</font></b><br/>
最近使ったプロジェクトファイルから選んで開きます。<br/>
<br/>
<b><font color='#000000'>終了</font></b><br/>
アプリケーションを終了します。編集中の内容がある場合は、保存の確認画面が表示されます。<br/>
</blockquote>

<h2>編集</h2>
<b><font color='#000000'>元に戻す</font></b><br/>
ひとつ前の作業状態に戻します。<br/>
<br/>
<b><font color='#000000'>やり直し</font></b><br/>
「元に戻す」を実行する前の状態に戻します。<br/>
<br/>
<b><font color='#000000'>切り取り</font></b><br/>
選択中のセリフや音符を切り取ります。<br/>
<br/>
<b><font color='#000000'>コピー</font></b><br/>
選択中のセリフや音符をコピーします。<br/>
<br/>
<b><font color='#000000'>貼り付け</font></b><br/>
ポジションカーソルの位置にコピー（切り取り）したセリフや音符を貼り付けます。<br/>
<br/>
<b><font color='#000000'>削除</font></b><br/>
選択中のセリフや音符を削除します。<br/>
<br/>
<b><font color='#000000'>すべて選択</font></b><br/>
編集中のトラックのセリフや音符すべてを選択状態にします。<br/>

<h2>表示</h2>
<b><font color='#000000'>標準</font></b><br/>
通常の表示です。<br/>
<br/>
<b><font color='#000000'>全画面</font></b><br/>
ウィンドウタイトルとメニューを非表示にして、編集画面を広くします。<br/>
※全画面を解除するには「F11」を押します。<br/>
<br/>
<b><font color='#000000'>コンパクト</font></b><br/>
キャラ画像のみにします。外部連携などに適した表示です。<br/>
※コンパクト表示中は再生／停止のほか、ポジションカーソル移動ショートカットキーも有効です。<br/>
※コンパクト表示中は、右クリックから「手前に表示」を切り替え可能です。<br/>
<br/>
<b><font color='#000000'>タイムベース</font></b><br/>
ツールバーやタイムラインの秒／小節表示を切り替えられます。<br/>
<br/>
<b><font color='#000000'>ツールバーの上下位置</font></b><br/>
ツールバーの表示位置を上段／中段／下段から選べます。<br/>
<br/>
<b><font color='#000000'>ツールバーの左右示位置</font></b><br/>
ツールバーの表示位置を左寄せ／中央／右寄せから選べます。<br/>
<br/>
<b><font color='#000000'>ツールバーの数字フォント</font></b><br/>
ツールバーの数字フォントをノーマル／ドットから選べます。<br/>

<h2>トラック</h2>
<b><font color='#000000'>トラックを追加</font></b><br/>
選択中のトラックの下に新しいトラックを追加します。<br/>
トークトラック、ソングトラック、オーディオトラックそれぞれ32トラックまで作成可能です。<br/>
<br/>
<b><font color='#000000'>トラックを複製</font></b><br/>
選択中のトラックを複製したトラックを追加します。<br/>
<br/>
<b><font color='#000000'>トラックを削除</font></b><br/>
選択中のトラックを削除します。<br/>
<br/>
<b><font color='#000000'>トラックを上に移動</font></b><br/>
選択中のトラックを1つ上に移動します。<br/>
<br/>
<b><font color='#000000'>トラックを下に移動</font></b><br/>
選択中のトラックを1つ下に移動します。<br/>
<br/>
<b><font color='#000000'>トラック内容をクリア</font></b><br/>
選択中のトラックの内容を空にします。<br/>
<br/>
<b><font color='#000000'>トラック内容を等間隔に整列</font></b><br/>
選択中のトラックの要素（セリフや外部オーディオ）を等間隔に配置します。<br/>
<br/>
<b><font color='#000000'>トラックをフリーズ</font></b><br/>
選択中のトラックの音声を固定（テンポラリファイルに出力）して、再生中のCPUとメモリの負荷を軽減します。複数トラックだと音が途切れたり、メモリ不足になる場合などに有効です。<br/>
※トラック内容を編集すると、フリーズは解除されます。<br/>
<br/>
<b><font color='#000000'>トラックをミュート</font></b><br/>
選択中のトラックを再生しません。<br/>
<br/>
<b><font color='#000000'>トラックをソロ</font></b><br/>
選択中のトラックのみ再生します。<br/>
<br/>
<b><font color='#000000'>トラックのキャスト</font></b><br/>
選択中のトークトラックまたはソングトラックのキャストを変更できます。<br/>

<h2>トーク</h2>
<b><font color='#000000'>単語の登録</font></b><br/>
正しく読めない単語の読みやアクセントを登録できます。<br/>
<br/>
<b><font color='#000000'>辞書の管理</font></b><br/>
登録した単語の一覧から、単語の編集や削除を行います。<br/>
<a href='https://cevio.jp/guide/cevio_ai/talktrack/talk_06/'>単語の登録と辞書の管理</a><br/>
<br/>
<b><font color='#000000'>セリフクリップ一覧</font></b><br/>
登録したセリフの一覧からセリフを選択行に挿入したり、一覧から削除できます。<br/>
<a href='https://cevio.jp/guide/cevio_ai/talktrack/phrasececlip/'>セリフクリップ</a><br/>
<br/>
<b><font color='#000000'>文のまとめ入力</font></b><br/>
1行ずつ入力すると手間のかかる長文をまとめて入力できます。セリフの挿入機能としても使えます。<br/>
<br/>
<b><font color='#000000'>検索</font></b><br/>
キーワードでセリフを検索します。セリフの行数が多い場合などに役立ちます。<br/>
<a href='https://cevio.jp/guide/cevio_ai/talktrack/search/'>セリフの検索</a><br/>
<br/>
<b><font color='#000000'>連続でセリフ入力</font></b><br/>
オンにすると、連続したセリフ入力を補助する以下の操作が可能になります。<br/>
<blockquote style='margin:0 0 0 40px;padding:0px;border:currentcolor'>
・[Enter]キーで編集確定すると自動的に次行の編集に移ります。<br/>
・[Shift + Enter]キーで編集確定すると、前行の編集に移ります。<br/>
・[Ctrl + Enter]で編集確定するか[Esc]でキャンセル、または空白行に[Enter]で入力を終えます。<br/>
</blockquote>
<br/>
<b><font color='#000000'>セリフの自動整列</font></b><br/>
オンにすると、セリフ入力に合わせて後続のセリフが自動的に前後に移動（間隔を維持）します。<br/>
オフにすると、セリフの編集や移動を行っても他のセリフの位置は変わりません。<br/>
<br/>
<b><font color='#000000'>空白行の追加</font></b><br/>
セリフリストの末尾に10行追加します。<br/>
<br/>
<b><font color='#000000'>音素グラフ</font></b><br/>
<blockquote style='margin:0 0 0 40px;padding:0px;border:currentcolor'>
<b><font color='#000000'>アクセント／音の高さ／音の大きさを調整</font></b><br/>
ACC／PIT／VOLの各調整モードに切り替えます。<br/>
<br/>
<b><font color='#000000'>長さのラインを重ねて表示</font></b><br/>
オフにすると、長さ調節のラインを非表示にできます。<br/>
※長さは音素グラフの上下の文字エリアのドラッグやマウスホイール上下でも調整可能です。<br/>
<br/>
<b><font color='#000000'>音素単位で調整</font></b><br/>
オンにすると、音素単位で細かく表示して調整できます。<br/>
オフのカナ単位表示でも、Alt+ドラッグで音素単位の調整が可能です。<br/>
<br/>
<b><font color='#000000'>初期値に戻す</font></b><br/>
選択中のセリフのアクセントや音の高さ／大きさ／長さのすべてを調整前の状態に戻します。<br/>
</blockquote>
<br/>
<b><font color='#000000'>音素グラフの配置</font></b><br/>
音素グラフの表示位置を変更できます。なしで非表示にすることもできます。<br/>
<br/>
<b><font color='#000000'>別トラックのセリフを表示</font></b><br/>
オフにすると、セリフリストに選択中のトラックのセリフのみ表示します。<br/>
<br/>
<b><font color='#000000'>列の表示</font></b><br/>
セリフリストのキャスト／開始時間／終了時間／長さ／トラック列の表示／非表示を変更します。<br/>

<h2>ソング</h2>
<b><font color='#000000'>編集ツール</font></b><br/>
選択ツール／まとめ選択ツール／ペンツール／ラインツール／消しゴムツールを切り替えます。<br/>
<a href='https://cevio.jp/guide/cevio_ai/operation/edittool/'>編集ツール</a><br/>
<br/>
<b><font color='#000000'>クオンタイズ</font></b><br/>
クオンタイズを変更します。<br/>
<a href='https://cevio.jp/guide/cevio_ai/operation/infopanel/'>クオンタイズとは</a><br/>
<br/>
<b><font color='#000000'>調整モード</font></b><br/>
音符や歌詞を入力する楽譜編集画面と、タイミングやピッチ等の調整画面を切り替えます。<br/>
「ビブラート無効」で、選択中のトラックのビブラートの振幅と周期を無効化することもできます。<br/>
<a href='https://cevio.jp/guide/cevio_ai/songtrack/song_07/'>調整画面</a><br/>
<br/>
<b><font color='#000000'>歌詞のまとめ入力</font></b><br/>
選択中の音符（ないときは先頭）から歌詞をまとめて入力します。<br/>
<br/>
<b><font color='#000000'>小節の挿入</font></b><br/>
選択中のトラックまたは全ソングトラックを対象に、指定した位置と長さで小節を挿入します。<br/>
末尾の追加は、ピアノロールの右端でスクロールバーの右スクロールボタン、Shift+マウスホイール下でも可能です。<br/>
<br/>
<b><font color='#000000'>小節の削除</font></b><br/>
選択中のトラックまたは全ソングトラックを対象に、指定した位置と長さで小節を削除します。<br/>
<br/>
<b><font color='#000000'>休符の除去</font></b><br/>
MIDIインポートなどで生じた細かい隙間（休符）を除去します。<br/>
<br/>
<b><font color='#000000'>音素で歌詞入力</font></b><br/>
歌詞を音素記号（発音記号）で入力します。<br/>
オンにすると音素入力モードになり、歌詞入力欄の背景がブルーグレーに変わります。<br/>
<br/>
<b><font color='#000000'>英語歌詞の自動分割</font></b><br/>
英語で歌詞入力したとき、1音符1音節に自動的に分割して割り当てます。<br/>
※歌詞表記の分割位置は自動推定によるため、正しくない場合があります。<br/>
<br/>
<b><font color='#000000'>自動まとめ選択</font></b><br/>
音符のドラッグ移動時、範囲内に調整があれば自動的にまとめ選択します。<br/>
※垂直移動はピッチ、横移動はタイミング以外の調整が範囲内にあるとき発動します。<br/>
<br/>
<b><font color='#000000'>調号は全トラック共通</font></b><br/>
オフにすると、トラック毎に調号を設定できます。オンにすると、選択中のトラックの調号を他のトラックにも反映して、調号が全ソングトラックで同期します（初期設定）。この設定はプロジェクトごとに保存されます。<br/>
<br/>
<b><font color='#000000'>別トラックの音符表示</font></b><br/>
オフにすると、楽譜編集画面に選択中のトラックの音符のみ表示します。<br/>
<br/>
<b><font color='#000000'>ルーラー表示</font></b><br/>
テンポ、拍子、調号（キー）、強弱記号の表示／非表示を変更します。<br/>
<br/>
<b><font color='#000000'>ライン表示</font></b><br/>
<blockquote style='margin:0 0 0 40px;padding:0px;border:currentcolor'>
<b><font color='#000000'>拍のライン／クオンタイズのライン</font></b><br/>
ピアノロールの拍やクオンタイズのラインの表示／非表示を変更できます。<br/>
<br/>
<b><font color='#000000'>調整画面にもライン表示</font></b><br/>
調整画面にも拍やクオンタイズのラインを表示します。<br/>
<br/>
<b><font color='#000000'>タイミングの状態ライン</font></b><br/>
タイミング調整画面で状態ラインを表示します。[Alt]キー押下で、一時的な表示切替も可能です。<br/>
</blockquote>
<br/>
<b><font color='#000000'>ガイドカーソル</font></b><br/>
マウスカーソルのクオンタイズ位置の表示／非表示を変更できます。<br/>
※楽譜編集画面ではペンツール／ラインツール／まとめ選択ツールのとき、調整画面では選択系ツールのとき表示されます。<br/>

<h2>トランスポート</h2>
<b><font color='#000000'>再生／停止</font></b><br/>
ポジションカーソルの位置から再生を開始します。<br/>
<br/>
<b><font color='#000000'>停止</font></b><br/>
再生を停止します。停止中は再生開始位置、先頭の順にポジションカーソルを移動します。<br/>
<br/>
<b><font color='#000000'>停止時に開始位置に戻る</font></b><br/>
再生停止時にポジションカーソルを再生開始位置に戻します。<br/>
<br/>
<b><font color='#000000'>再生中の自動スクロール</font></b><br/>
再生中のタイムライン／セリフリスト／ピアノロールを自動スクロールします。<br/>
オフにすると、再生中も手動スクロールできるようになります。再生中も変更可能です。<br/>
<br/>
<b><font color='#000000'>繰り返し</font></b><br/>
再生の繰り返しを設定／解除します。<br/>
<br/>
<b><font color='#000000'>始点マーカー</font></b><br/>
繰り返しの始点を設定／解除します。<br/>
<br/>
<b><font color='#000000'>終点マーカー</font></b><br/>
繰り返しの終点を設定／解除します。<br/>
<br/>
<b><font color='#000000'>先頭へ移動</font></b><br/>
ポジションカーソルを先頭へ移動します。<br/>
<br/>
<b><font color='#000000'>始点マーカーへ移動</font></b><br/>
ポジションカーソルを始点マーカーの位置へ移動します。<br/>
<br/>
<b><font color='#000000'>終点マーカーへ移動</font></b><br/>
ポジションカーソルを終点マーカーの位置へ移動します。<br/>
<br/>
<b><font color='#000000'>選択要素へ移動</font></b><br/>
ポジションカーソルを選択中のセリフや外部オーディオの左端に移動します。<br/>
<br/>
<b><font color='#000000'>指定位置へ移動</font></b><br/>
ポジションカーソルを指定位置へ移動します。<br/>
「分：秒」または「小節：拍」で位置を入力できます。<br/>
<br/>
<b><font color='#000000'>カーソルへ移動</font></b><br/>
ポジションカーソルの位置を表示します。ポジションカーソルを画面外で見失ったときなどに便利です。<br/>
