<h1>トークトラックの説明</h1>
<br/>
<div class="main_text">
左側のセリフリストにテキストを入力して、右側のコンディションや感情コントロール、音素グラフで調整を行います。<br/>
メニューの「トラック」やトラック上の右クリックから「トークトラックを追加」を選ぶと、32トラックまで増やすことができます。<br/>
<br/>
<div><img src='../image/001_w.png' border='0'/></div>
<br/>
<b><font color='#000000'>試聴</font></b><br/>
セリフリストで選択中の行を試聴します。試聴中にもう一度押すと中断します。<br/>
音素グラフを左クリックして、指定位置から試聴することも可能です。<br/>
<br/>
<b><font color='#000000'>繰り返し試聴</font></b><br/>
試聴の繰り返しを設定／解除します。<br/>
<br/>
<b><font color='#000000'>自動試聴</font></b><br/>
セリフ入力後や、コンディション／感情／音素グラフを調整したとき、自動的に試聴するかどうかを切り替えられます。<br/>
オプションの「音素グラフの自動試聴で部分再生」をチェックすると、音素グラフをの自動試聴で、調整箇所の直前のポーズ(句読点等)から直後のポーズまでを再生します。<br/>

<blockquote style='margin:0 0 0 40px;border:none;padding:0px'>
<div class="box30">
<div class="box-title">自動試聴の活用</div>
<p>
セリフ入力中に[Enter]キーを押すと自動試聴できるため、「連続でセリフ入力」と組み合わせて、あらかじめ台本を入力しておいて順番に再生したり、生放送などでリアルタイム音声合成ツールとしても活用できます。
</p>
</div>
</blockquote>

<h2>右クリックメニュー</h2>
セリフリストで右クリックすると便利なメニューが表示されます。<br/>
<br/>
<div><img src='../image/talk_contextmenu.png' border='0'/></div>
<br/>
<blockquote style='margin:0px 0px 0px 40px;padding:0px;border:currentcolor'>
<b><font color='#000000'>切り取り</font></b><br/>
選択中のセリフを切り取ります。<br/>
<br/>
<b><font color='#000000'>コピー</font></b><br/>
選択中のセリフをコピーします。<br/>
<br/>
<b><font color='#000000'>貼り付け</font></b><br/>
選択行にコピー（切り取り）したセリフを貼り付けます。<br/>
<br/>
<b><font color='#000000'>パラメータのみ貼り付け</font></b><br/>
選択行にコピー（切り取り）したセリフのパラメータのみ貼り付けます。※セリフのコピー後のみ表示されます。<br/>
<br/>
<b><font color='#000000'>コピーした内容の挿入</font></b><br/>
選択行の位置にコピーしたセリフを挿入します。※セリフのコピー後のみ表示されます。<br/>
<br/>
<b><font color='#000000'>挿入</font></b><br/>
選択行の位置に空白行を挿入します。<br/>
<br/>
<b><font color='#000000'>削除</font></b><br/>
選択行を削除します。<br/>
<br/>
<b><font color='#000000'>移動</font></b><br/>
選択行のセリフを指定の方法で移動します。<br/>
<br/>
<b><font color='#000000'>文のまとめ入力</font></b><br/>
文章をまとめて入力して選択行に挿入します。<br/>
<br/>
<b><font color='#000000'>検索</font></b><br/>
キーワードでセリフを検索します。<br/>
<br/>
<b><font color='#000000'>連続でセリフ入力</font></b><br/>
オンのとき、セリフの編集確定後、自動的に次行の編集に移ります。<br/>
<br/>
<b><font color='#000000'>表示</font></b><br/>
<blockquote style='margin:0px 0px 0px 40px;padding:0px;border:currentcolor'>
<b><font color='#000000'>別トラックのセリフ</font></b><br/>
オンのとき、セリフリストに選択中のトラック以外のセリフも表示します。<br/>
<br/>
<b><font color='#000000'>キャスト／開始時間／終了時間／長さ／トラック</font></b><br/>
各列の表示／非表示を切り替えられます。<br/>
</blockquote>
<br/>
<b><font color='#000000'>ここから再生</font></b><br/>
ポジションカーソルの位置に関係なく、選択中のセリフから再生を開始できます。<br/>
ショートカットキー[F5]で再生停止すると、停止位置のセリフ先頭にポジションカーソルを移動して選択します。<br/>
[F5]キーをスペースキーの代わりに使ったり、セリフ単位で読み進めたりできます。<br/>
<br/>
<b><font color='#000000'>WAV書き出し</font></b><br/>
選択行のセリフをモノラルWAV形式でファイル出力します。<br/>
※サンプリングレートとビット深度は<a href='https://cevio.jp/guide/cevio_ai/option/'>オプション</a>で指定可能。<br/>
<br/>
<b><font color='#000000'>セリフクリップの挿入</font></b><br/>
セリフクリップに登録済みのセリフから選んで、選択行に挿入します。<br/>
<br/>
<b><font color='#000000'>セリフクリップの登録</font></b><br/>
セリフクリップに選択行のセリフを登録します。<br/>
<br/>
<b><font color='#000000'>単語の登録</font></b><br/>
選択行のセリフを単語登録します。<br/>
</blockquote>
