<h1>セリフの入力と編集</h1>
<br/>
<div class="main_text">
<h2>セリフの入力</h2>
行をクリックすると選択、選択行をクリックすると入力状態になります。<br/>
入力状態でセリフを入力したら、[Enter]キーで確定します。<br/>
キーボード操作のみでも、[Enter]キーや下カーソルキーで次行、上カーソルキーで前行に選択行を移動して、[F2]キーで選択行の入力ができます。<br/>
（非入力状態から[F2]キーを押さずに、そのまま文字入力することも可能です。）<br/>
<br/>
日本語ボイスのセリフは、全角のひらがな／カタカナ／漢字などを使った自然な文章で入力できます。<br/>
英語ボイスのセリフは、半角のアルファベットや数字などを使った英文で入力できます。<br/>
ひとつのセリフ欄には、日本語ボイスは200文字、英語ボイスは500文字まで入力可能です。<br/>
<br/>
<div><img src='../../image/lt_02_w.png' border='0'/></div>

<blockquote style='margin:0 0 0 40px;border:none;padding:0px'>
<div class="box30">
<div class="box-title">日本語セリフ入力のヒント</div>
<p>
発音はセリフ1行の文全体から決まります。文末の疑問符(？)や感嘆符(！)も発音に反映されます。<br/>
読点(、)のない長い文や、1文を複数行に細切れに入力すると正しく発音できません。<br/>
「こんにちは。○○さん。」のような、1行の中の句点(。)は読点に置き換えて読み上げられます。<br/>
同じ読みでも同音異義語や、ひらがな・カタカナに変えてみると思い通りの発音が見つかることがあります。<br/>
難しい人名や地名は、<a href='../talk_06/'>単語登録</a>しておくと正しく発音できます。<br/>
</p>
</div>
</blockquote>

<b><font color='#000000'>読み指定カッコの挿入</font></b><br/>
セリフ入力中、右クリックメニューの「読み指定カッコの挿入」でルビ記号を入力できます。<br/>
<div><img border="0" src="../../image/lt_07_w_x.png"></div>
・日本語ボイスの場合、カーソル位置に《》を挿入します。<br/>
　テキスト選択状態なら、ルビ先頭を示す｜も挿入します。<br/>
・英語ボイスの場合、カーソル位置の単語の末尾に[]を挿入します。<br/>

<blockquote style='margin:0 0 0 40px;border:none;padding:0px'>
<div class="box30">
<div class="box-title">日本語ルビ入力（読み指定）</div>
<p>
《》内にひらがな／カタカナを書いて、読みを指定することができます。<br/>
（例）　今朝｜四月一日《わたぬき》さんに会った。　※「｜」でルビ対象を明示。<br/>
（例）　こんにちは、四月一日《わたぬき》さん。　　※「｜」を省略したときは始まりを推定。<br/>
《》内は<a href='../talk_06/'>単語登録</a>と同様に「’」で母音の無声化も指定できます。<br/>
</p>
</div>

<div class="box30">
<div class="box-title">英語ルビ入力（発音記号指定）</div>
<p>
単語に続けて[]内に発音記号を書くと、読みを指定することができます。<br/>
音節（シラブル）の区切りは半角縦棒「|」，音節内の音素の区切りはコンマ「,」で表します。<br/>
母音の後ろにはストレス（アクセント）を表す数字を付けます（省略すると0）。<br/>
0:ストレス無し 1:プライマリストレス 2:セカンダリストレス<br/>
（例）　Sasara[s,ah1|s,ah0|r,ah0]<br/>
（例）　Ando[ah1,n|d,ow0]<br/>
発音記号の一覧は<a href='../talk_01_english/'>こちら</a><br/>
</p>
</div>
</blockquote>

<b><font color='#000000'>連続でセリフ入力</font></b><br/>
右クリックメニューの「連続でセリフ入力」をオンにすると、セリフの連続入力モードになります。<br/>
テキスト入力と再生を連続して行いたいときなどに活用してください。<br/>
<div><img src='../../image/lt_08_w.png' border='0'/></div>
・[Enter]キーで編集確定すると自動的に次行の編集に移ります。<br/>
・[Shift + Enter]キーで編集確定すると、前行の編集に移ります。<br/>
・[Ctrl + Enter]で編集確定するか[Esc]でキャンセル、または空白行に[Enter]で入力を終えます。<br/>

<h2>セリフのキャスト設定</h2>
複数のキャストを使用可能な場合、キャスト欄でそのセリフを発声するキャラクターを選択することができます。<br/>
※「トラック」メニューの「キャスト」やトラックヘッダからキャストを変更すると、トラック内を一括で変更できます。<br/>
<br/>
<div><img src='../../image/lt_03_w.png' border='0'/></div>

<h2>切り取り／コピー／貼り付け</h2>
選択行をコピーしたり切り取って、別の行に貼り付けることができます。<br/>
右クリックのメニューや、ショートカットキー[Ctrl+X]（切り取り）、[Ctrl+C]（コピー）、[Ctrl+V]（貼り付け）で操作可能です。<br/>
コンディションや感情、音素グラフの内容も含めて貼り付けできるため、調整済みのセリフの使い回しにも便利です。<br/>
※「別トラックのセリフ表示」オンときも、選択トラックのみ対象に切り取り／コピーを行います。<br/>
※セリフ入力中以外は複数行の貼り付けにも対応し、別のソフトでコピーしたテキストも貼り付け可能です。<br/>
<br/>
<div><img src='../../image/tk01_03_w.png' border='0'/></div>
<br/>
<b><font color='#000000'>パラメータのみ貼り付け</font></b><br/>
セリフをコピーまたは切り取り後、パラメータを適用したい行(複数選択可)で右クリックして「パラメータのみ貼り付け」を選ぶと、セリフは貼り付けずに速さや高さ・感情などのパラメータのみ反映します。<br/>
※通常の貼り付け同様、空行にもパラメータは反映されます。<br/>
<div><img border="0" src="../../image/lt_09_w.png"></div>

<h2>複数行の選択</h2>
1行選択してから、[Shift]キーを押しながら別の行をクリックすると、複数行を選択することができます。<br/>
また、[Ctrl+A]ですべての行を選択することも可能です。<br/>
複数行の切り取りやコピーができるほか、複数行にプリセットを適用することも可能です。<br/>
※キャストが一致する選択行にプリセットは適用されます。<br/>

<h2>行の削除</h2>
[Delete]キーまたは右クリックのメニューの「削除」で、選択行を削除することができます。<br/>

<h2>行の挿入</h2>
右クリックのメニューの「挿入」で、選択行の位置に行を挿入できます。<br/>
切り取り／コピー中はその行を、それ以外は空白行が挿入されます。<br/>
別のソフトでコピーしたクリップボードのテキストも挿入できます。<br/>
※空白行を挿入したいときは、「Esc」キーでクリップボードをクリアできます。<br/>

<h2>空白行の追加</h2>
メニューの「トーク」の「空白行の追加」で、セリフリストの末尾に空白行を10行追加可能です。<br/>

<h2>表示の変更</h2>
右クリックのメニューの「表示」からは、リストの表示内容を変更することができます。<br/>
「別のトラックのセリフ表示」をチェックすると、別トラックのセリフもリストに表示できます。<br/>
「セリフ、開始時間、終了時間、長さ」では、各列の表示／非表示を切り替えられます。<br/>
<br/>
<div><img src='../../image/tk01_05_w.png' border='0'/></div>
<br/>
<blockquote style='margin:0 0 0 40px;border:none;padding:0px'>
<b><font color='#000000'>別トラックのセリフ</font></b><br/>
セリフリストに、選択中のトラック以外のセリフも表示します。<br/>
<br/>
<b><font color='#000000'>キャスト</font></b><br/>
セリフを発声するキャラクター。<br/>
<br/>
<b><font color='#000000'>開始時間</font></b><br/>
セリフが開始する時間（分:秒.ミリ秒）。直接時間を入力してセリフを移動することもできます。<br/>
<br/>
<b><font color='#000000'>終了時間</font></b><br/>
セリフが終了する時間（分:秒.ミリ秒）。1行ずつ経過時間を把握することができます。<br/>
<br/>
<b><font color='#000000'>長さ</font></b><br/>
セリフの長さ（分:秒.ミリ秒）。<br/>
<br/>
<b><font color='#000000'>トラック</font></b><br/>
セリフのトラック。セリフ入力後に別トラックを選択すると、セリフを簡単にトラック移動できます。<br/>
</blockquote>
<br/>
※キャストとトラックを非表示、開始時間と長さを表示した例。
